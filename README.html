        Description

        The RAW Benchmark Suite has been designed to facilitate benchmarking and comparing reconfigurable computing systems. This Suite has the following features:

        • Benchmarks are portable to any reconfigurable computer as a behavioral Verilog netlist with a single address space (memory-mapped) external interface to an included C driver program.
        • Benchmarks include a subset from the existing pool of CPU and parallel processing benchmarks.
        • Benchmarks are small and easy to understand.
        • Benchmarks are parameterized to generate designs from one to several hundred FPGAs (millions of design gates).
        • Each benchmark includes a C version that, under ideal circumstances (i.e. when such a compiler is developed) can be compiled into the Verilog version.

        Reference Results

        These benchmark have been compiled and tested on an IKOS VirtuaLogic Emulator (software version 1.4.3) connected to a host workstation. The release contains reference results comparing performance to a 6.6 SPECint95 Sun Ultra Enterprise 3000 (167MHZ processor) using gcc -O4.

        Caveat

        These benchmarks are not designed to serve as a standard and have not been selected to represent any meaningful subset of actual programs or applications.
