Synopsys to IKOS VirtuaLogic Emulation System

        Directory Contents

        The following Synopsys Behavioral Compiler scripts target the VirtuaLogic reference library:

        • verilog2vle.syn - Behavioral Verilog to VirtuaLogic script
        • vhdl2vle.syn - Behavioral VHDL to VirtuaLogic script
        • flatten_vle.syn - Clean pass of Verilog through Synopsys

        The following Synopsys library is for the VirtuaLogic 1.4 Reference primitives:

        • vmw_ref.db - VirtuaLogic 1.4 Reference Synthesis Library for Synopsys
